# Coleções (mosaicos Bento)

Imagens de fundo dos mosaicos das coleções. Recomendado **1200×900 (4:3)**.
O mosaico do Pokémon é o maior (destaque) — usa a melhor imagem aqui.

| Ficheiro       | Coleção             |
| -------------- | ------------------- |
| `pokemon.png`  | Pokémon (destaque)  |
| `magic.png`    | Magic: The Gathering|
| `yugioh.png`   | Yu-Gi-Oh!           |
| `onepiece.png` | One Piece Card Game |
| `lorcana.png`  | Disney Lorcana      |

Há sempre uma camada escura por cima para garantir leitura do texto.
Se faltar algum ficheiro, fica o gradiente da respetiva coleção.
